⚙️ Конфигурационный файл калькулятора

Файл calculator_config.json позволяет настраивать внешний вид, размеры окна, шрифты и цветовые темы приложения без перекомпиляции.

ℹ️ О программе ⌨️ Горячие клавиши

📁 Расположение файла

Приложение ищет calculator_config.json в следующих местах (по порядку):

  1. Рядом с исполняемым файлом (bin/Debug/net8.0/)
  2. В корневой папке проекта (../../../)
  3. В папке Resources/Configs/ проекта

Если файл не найден или содержит ошибки, приложение использует настройки по умолчанию и показывает предупреждение.

💡 Совет: Для разработки удобно держать файл в корне проекта. При публикации поместите его рядом с .exe.

📋 Структура конфигурации

🔷 Корневой объект

Параметр Тип Описание Значение по умолчанию
defaultTheme string Название темы, которая будет применена при запуске "dark"
window object Настройки окна приложения см. таблицу ниже
fonts object Настройки шрифтов см. таблицу ниже
themes object Словарь цветовых тем (ключ — имя темы) "dark" и "light"

🪟 Объект window

Параметр Тип Описание Значение по умолчанию
width double Ширина окна при запуске (пиксели) 320
height double Высота окна при запуске (пиксели) 500
minWidth double Минимальная ширина окна 250
minHeight double Минимальная высота окна 400
title string Заголовок окна "Calculator"

🔤 Объект fonts

Параметр Тип Описание Значение по умолчанию
buttonFontFamily string Название шрифта для кнопок "Segoe UI"
buttonFontSize double Размер шрифта кнопок (пиксели) 24
displayFontSize double Размер шрифта дисплея (пиксели) 48
historyFontSize double Размер шрифта истории 16
menuFontSize double Размер шрифта меню 14

🎨 Объект themes

Каждая тема имеет ключ (например, "dark") и объект с полями:

Параметр Тип Описание
displayName string Название темы для отображения в меню
description string Краткое описание темы
colors object Словарь цветов в формате HEX (ключ → значение #RRGGBB)

📝 Пример полного конфигурационного файла

{ "defaultTheme": "dark", "window": { "width": 360, "height": 540, "minWidth": 280, "minHeight": 420, "title": "Мой калькулятор" }, "fonts": { "buttonFontFamily": "Consolas", "buttonFontSize": 22, "displayFontSize": 44, "historyFontSize": 14, "menuFontSize": 12 }, "themes": { "dark": { "displayName": "Тёмная тема", "description": "Ночной режим", "colors": { "WindowBackgroundColor": "#1E1E1E", "DigitButtonBackgroundColor": "#2D2D2D", "DigitButtonForegroundColor": "#FFFFFF" } }, "light": { "displayName": "Светлая тема", "description": "Дневной режим", "colors": { "WindowBackgroundColor": "#F5F5F5", "DigitButtonBackgroundColor": "#FFFFFF", "DigitButtonForegroundColor": "#1E1E1E" } } } }

🔄 Как изменения повлияют на приложение

  • Изменение defaultTheme — приложение запустится с указанной темой.
  • Изменение размеров окна — окно калькулятора откроется с новыми размерами.
  • Изменение шрифтов — все кнопки, дисплей и меню изменят шрифт и размер.
  • Добавление новой темы — новая тема появится в меню выбора.
  • Изменение цветов — интерфейс перекрасится в соответствии с новыми HEX-кодами.
⚠️ Важно: HEX-коды должны начинаться с символа # (например, #FF5733). Приложение игнорирует лишние запятые и комментарии в JSON.
