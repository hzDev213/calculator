🧮 Калькулятор на WPF v1.0

Современный калькулятор с горячих клавиш и сменой тем.
Разработан с использованием паттернов MVVM и Decorator.

⌨️ Горячие клавиши

⚙️ Технологический стек

.NET 8+ WPF C# 12 MVVM Decorator Pattern Factory Method

📐 Поддерживаемые операции

  • Арифметика: +, -, ×, ÷
  • Константы: π (пи) — ввод с кнопки или через P → I

🏗️ Архитектура проекта

Проект строго разделён на слои в соответствии с паттерном MVVM:

  • View (MainWindow.xaml) — XAML-разметка, стили, привязки
  • ViewModel (MainWindowViewModel) — логика представления, команды, состояние
  • Model — бизнес-логика (CalculatorEngine), декораторы, фабрики кнопок

🧩 Ключевые паттерны

ПаттернГде используетсяНазначение
DecoratorEnginePiDecoratorДинамическое добавление обработки константы π без изменения основного движка.
Factory MethodCommonButtonFactoryЦентрализованное создание всех типов кнопок (цифровые, операторы, константы).

🎨 Возможности интерфейса

  • ✅ Смена темы: Тёмная / Светлая (через меню-гамбургер)
  • ✅ Адаптивный дизайн — окно масштабируется без потери функциональности
  • ✅ Тактильная обратная связь: анимация нажатия кнопок
  • ✅ Звуковое сопровождение: раздельные звуки для обычных кнопок и "="
